# 新生入学管理系统 python886

## 项目概述

新生入学管理系统是一个基于Python语言的Web应用，结合了Django框架的后端逻辑以及Vue.js的前端界面设计，致力于简化和优化新生入学的管理流程。

## 技术栈

- **后端**：Python 3.7，Django框架
- **前端**：Vue.js
- **数据库**：MySQL
- **开发工具**：PyCharm

## 功能模块

### 系统角色

- **管理员**
- **用户**

### 管理员功能

- **首页**：展示系统概况
- **个人中心**：个人信息管理
- **用户管理**：用户账户的管理与权限分配
- **新生信息管理**：管理新生资料
- **宿舍安排管理**：安排新生住宿
- **系统管理**：系统设置与维护

### 用户功能

- **首页**：快速导航到各功能模块
- **个人中心**：管理个人信息
- **新生信息管理**：维护个人入学信息
- **宿舍安排管理**：查看宿舍安排信息

## 运行环境

- Python 3.7 或以上版本
- MySQL 数据库环境
- Node.js（用于前端Vue.js）

## 部署步骤

1. 配置Python环境（3.7版本）
2. 安装MySQL数据库，并建立相应的数据表
3. 使用PyCharm或命令行部署Django后端应用
4. 构建Vue.js前端项目，并部署到Web服务器

- **基于Python+Django的高效后端**：快速开发，高性能，易于维护
